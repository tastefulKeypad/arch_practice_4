# Проектирование документной модели

## Сущности
Были выделены следующие сущности:
| Сущность     | Коллекция | Описание                     |
|--------------|-----------|------------------------------|
| Автомобиль   | cars      | Автомобили из автопарка      |
| Пользователь | users     | Зарегистрированные клиенты   |
| Аренда       | rents     | Записи об аренде автомобилей |

## Коллекции
Коллекция автомобилей:
```javascript
"_id": objectId,    // Уникальный ID автомобиля, генерируется автоматически
"carClass": int,    // Класс автомобиля
"price": int,       // Стоимость автомобиля
"capacity": int,    // Вместимость автомибиля
"name": string      // Название автомобиля
```
Коллекция пользователей:
```javascript
"_id": objectId,    // Уникальный ID пользователя, генерируется автоматически
"email": string,    // Почта пользователя
"name": string,     // Имя пользователя
"surname": string,  // Фамилия пользователя
"password": string, // Хэшированный пароль пользователя
"isAdmin": bool     // Флаг администратора
```
Коллекция аренд:
```javascript
"_id": objectId,    // Уникальный ID аренды, генерируется автоматически
"carId": objectId,  // Reference на ID арендованного автомобиля
"userId": objectId, // Reference на ID арендатора
"dateStart": date,  // Дата начала аренды
"dateEnd": date,    // Дата окончания аренды
"status": string    // Статус аренды
```
